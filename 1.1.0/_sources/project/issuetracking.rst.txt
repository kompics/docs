Issue Tracking
--------------
Kompics is using GitHub Issues as its issue tracking system.

Browsing
^^^^^^^^
Tickets
"""""""
Before filing a ticket, please check the existing `Kompics tickets <https://github.com/kompics/kompics/issues>`_ for earlier reports of the same problem.

You are very welcome to comment on existing tickets, especially if you have reproducible test cases that you can share.

Creating tickets
""""""""""""""""
Please include the versions of Java and Kompics and relevant configuration files.

You can create a `new tickets <https://github.com/kompics/kompics/issues/new>`_ if you have registered a GitHub user account.

Thanks a lot for reporting bugs and suggesting features!
